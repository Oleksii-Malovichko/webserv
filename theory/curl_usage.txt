curl - утилита для отправки HTTP/HTTPS (и других протоколов) запросов к серверу прямо из терминала

curl http://localhost		отправка GET-запроса, выводит тело ответа

curl -I http://localhost	отправка HEAD-запроса, получает только заголовки ответа, без тела страницы

curl -v http://localhost	включает подробный режим, показывает все, что происходит "под капотом"
{
	соединение с сервером
	отправленные HTTP-заголовки
	полученные HTTP-заголовки
	тело ответа
} (полезно для отладки если что-то не работает)

Доп полезные опции:
-X <METHOD> -> задает HTTP-метод (GET, POST, DELETE, PUT) (тест разных HTTP методов)
-d "key=value" -> отправка данных POST-запросом (тест API или формы)
-Р "header: value" -> добавить кастомный HTTP-заголовок
-L -> следовать за редиректами (если сервер перенаправляет на другой URL)
